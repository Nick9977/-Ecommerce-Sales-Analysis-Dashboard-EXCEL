# Ecommerce Sales Analysis Dashboard | Excel

## Project Overview
This project presents an interactive Ecommerce Sales Analysis Dashboard created using Microsoft Excel. The dashboard helps analyze sales performance, profit trends, customer behavior, and product performance through dynamic visualizations and business insights.

## Tools & Technologies Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Conditional Formatting
- Data Cleaning
- Dashboard Design

## Key Insights
- Monthly Sales and Profit Analysis
- Top Selling Products
- Region-wise Sales Performance
- Customer Purchase Trends
- Profitability Analysis
- KPI Tracking

## Dashboard Features
- Interactive Filters & Slicers
- Dynamic Charts
- Automated Calculations
- User-Friendly Dashboard Layout
- Business Performance Reporting

## Skills Demonstrated
- Data Analysis

- Dashboard Development
- Business Intelligence
- Reporting & Insights Generation
